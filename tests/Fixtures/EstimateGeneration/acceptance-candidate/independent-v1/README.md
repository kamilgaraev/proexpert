# Кандидат независимого acceptance-корпуса МОСТ

Статус: `pending_owner_approval`.

Корпус нельзя использовать для acceptance gate до независимой проверки исходников, эталонов и recorded envelopes владельцем продукта. Эталоны находятся в `cases/*/expected.json` и не генерируются из predictions. `manifest.json` фиксирует SHA-256 каждого исходника и эталона.

Состав: свободный эскиз, сканированный PDF, векторный PDF, DXF, настоящий DWG R2000 и инженерный эскиз с конфликтующим масштабом.
