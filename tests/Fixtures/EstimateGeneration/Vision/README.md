# Geometry fixtures

`simple-house.dxf` is an original synthetic R2000 fixture authored for МОСТ. It contains a millimetre drawing with LINE, closed LWPOLYLINE and TEXT entities.

`simple-house.dwg` was generated only from that fixture with the official GNU LibreDWG 0.13.4 `dxf2dwg` executable:

```text
dxf2dwg -o simple-house.dwg simple-house.dxf
```

No third-party drawing content is included. LibreDWG is GPL-3.0-or-later; the fixture is test data generated by the separate CLI and is not linked with application code.
