# Склад, объект, ответственное хранение и списание

## Термины

- Складское списание: уменьшение остатка из-за утилизации, порчи, утраты, инвентаризационной корректировки или производственного расхода.
- Выдача ответственному: передача материала прорабу или МОЛ под ответственность. Это не списание.
- Объектовый склад: реальный склад организации с типом `project`, привязанный к проекту.
- Склад ответственного: скрытый склад организации с типом `custody`, привязанный к проекту и пользователю.
- Производственный расход: списание материала из склада ответственного при сохранении записи в журнале работ.

## Цепочка

1. Центральный склад получает материал.
2. Материал распределяется и отправляется на объект.
3. Объект принимает материал на объектовый склад.
4. Материал выдается ответственному.
5. Ответственный использует материал в журнале работ.
6. Использованный материал списывается в расход.

## Инварианты

- Нельзя отправить на объект больше доступного остатка исходного склада.
- Нельзя принять на объект больше отправленного количества.
- Нельзя выдать ответственному больше доступного остатка объектового склада.
- Нельзя списать через журнал больше остатка у ответственного.
- Возврат ответственным увеличивает объектовый склад и уменьшает склад ответственного.
- `write_off` без журнала работ используется только для потерь, брака, утилизации и корректировок.
