# Автоматический выбор актуальной ФСНБ

AI-сметчик для каждой новой сессии автоматически выбирает последнюю успешно импортированную версию `fsnb_2022`. Рабочей считается версия со статусом `parsed`, завершённым импортом, ненулевым числом импортированных строк и отсутствием ошибок.

Версии в состояниях `importing` и `failed`, пустые и частично импортированные версии не участвуют в выборе. Если свежий импорт не прошёл проверку, используется предыдущая рабочая версия. Выбранная версия сохраняется в контексте сессии и не меняется в ходе расчёта.

Переменная `ESTIMATE_GENERATION_NORM_APPROVED_DATASET_VERSION` больше не управляет рабочей версией. Изменение ограничено модулем AI-сметчика и не влияет на обычные сметы.
