# Plan 4 — Task 5: визуальная проверка геометрии

## Результат

Реализован полный контур визуальной проверки чертежей AI-сметчика: backend выдаёт актуальную модель здания и временные ссылки на постраничные изображения, а админка позволяет сверять геометрию, задавать масштаб и отправлять только разрешённые типизированные исправления с CAS-версиями.

Обычные сметы не изменялись.

## Backend

Коммиты:

- `87a90b20 feat[lk]: добавлен контракт проверки геометрии AI-сметы`
- `491f0f12 fix[lk]: добавлены источники эскизов AI-сметы`

Добавлен `GET /api/v1/admin/projects/{project}/estimate-generation/sessions/{session}/geometry`, который возвращает:

- `state_version`, `model_version`, `input_version`;
- последнюю нормализованную модель здания;
- источники по страницам с размерами и краткоживущими подписанными S3-ссылками;
- растрированные страницы PDF и исходные PNG/JPEG-эскизы строго в контексте организации и сессии.

Подтверждение геометрии использует существующий `POST .../geometry/confirm`. Поддержаны масштаб и закрытый набор операций `replace`; произвольные пути из пользовательского ввода не принимаются.

Проверки backend:

- целевые PHPUnit-тесты: 7/7, 25 assertions, PASS;
- `php -l` затронутых файлов: PASS;
- целевой Pint: PASS;
- целевой PHPStan: `[OK] No errors`.

## Frontend

Коммит:

- `26be3113 feat[lk]: добавлена проверка чертежей AI-сметы`

Реализованы:

- строгая нормализация ответа geometry review;
- загрузка изображения и модели с отменой устаревших запросов при смене сессии;
- SVG-слой поверх чертежа с корректной компенсацией letterbox для широких и высоких изображений;
- выбор элементов с клавиатуры, видимый фокус, подписи, уверенность и статус не только цветом;
- перевод точек масштаба в реальные пиксели источника и единицы м/см/мм;
- инспектор помещений, стен, проёмов и этажа с формированием только разрешённых путей;
- точный CAS-запрос с `state_version`, `model_version`, `input_version`;
- обработка `409`: обновление снимка и геометрии без автоматического повтора, с явным подтверждением пользователя;
- интеграция шага в рабочее пространство AI-сметчика ровно один раз;
- адаптивная компоновка без новой canvas-зависимости.
