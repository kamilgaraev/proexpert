# Task 6 — повторная проверка согласования документов

## Результат

Закрыты повторные замечания по согласованию юридических документов без сохранения устаревших веток и дублирующих прав.

## Исправления

- Введена единая проекция статуса согласования в approval_status и lifecycle_status; отправка, завершение и восстановление используют один источник правил.
- Разрешение контекста авторизации переведено на read-only lookup: проверка участника согласования больше не создаёт системные, организационные или проектные контексты.
- due_at экземпляра пересчитывается как минимальный срок активных шагов после каждого решения и сбрасывается у терминального процесса.
- Восстановление строит полную производную проекцию экземпляра, шагов, назначений, сроков и документа только из неизменяемого снимка и журнала решений; противоречивые доказательства отклоняются, ошибка аудита откатывает восстановление и сохраняет маркер.
- Для reject/return/reassign/cancel закреплены точные обязательные поля ввода и канонические ключи назначения target_actor_type/target_actor_id.
- PostgreSQL-миграции проверяют точные дескрипторы индексов и ограничений; несовпадающий валидный объект вызывает fail-closed, повреждённый незавершённый индекс восстанавливается, валидация работает по явному allowlist.
- Все изменения агрегата документа используют порядок document → file → version с транзакционной advisory-блокировкой PostgreSQL.
- Смена текущей версии запрещена при активном согласовании.
- PostgreSQL opt-in fixture использует реальные production-миграции и отдельные процессы для гонок версии шаблона, отправки, параллельных и терминальных решений, а также отправки против смены текущей версии.
- Удалено неиспользуемое право legal_archive.workflow.manage из backend, RBAC-конфигурации, переводов и тестовых контрактов.

## Проверки

- git diff --check — успешно.
- Синтаксис всех изменённых PHP-файлов — успешно.
- Pint для всех изменённых PHP-файлов — успешно.
- PHPStan для изменённого backend-кода и миграций с лимитом 1 ГБ — ошибок нет. Первый запуск с лимитом 128 МБ завершился исчерпанием памяти и был корректно повторён.
- PHPUnit: LegalWorkflowArchitectureTest, LegalArchivePermissionTranslatorTest, LegalDocumentWorkflowTest, LegalDocumentVersionConcurrencyTest — 45 тестов, 313 проверок, успешно.
- PostgreSQL process-race suite локально не запускался: он намеренно требует выделенную тестовую PostgreSQL-БД и явное разрешение DDL; запуск DB-команд в локальном окружении запрещён условиями задачи.
